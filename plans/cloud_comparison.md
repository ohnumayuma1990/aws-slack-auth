# 他クラウドサービスとの比較

AWS以外の主要クラウドサービスでの無料枠と実現可能性について。

## 1. Google Cloud Platform (GCP)
- **Cloud Functions**: 月間200万呼び出しまで無料枠あり。
- **Cloud Run**: コンテナベースのため、ODBCドライバのインストールがDockerfileで容易に行えます。無料枠も強力です。
- **実現性**: 非常に高い。特にODBCドライバの依存関係解決において、コンテナが使えるCloud Runが有利です。

## 2. Microsoft Azure
- **Azure Functions**: 月間100万実行まで無料（従量課金プラン）。
- **ODBC利用**: WindowsベースのプランであればネイティブにODBCが扱いやすい場合もありますが、Linuxベースでもカスタムコンテナの使用が可能です。
- **実現性**: 高い。Active Directoryとの親和性が高いですが、Slack認証のみであればAWS/GCPと同様です。

## まとめ
- **コスト面**: いずれのクラウドも、小規模な認証リダイレクト処理であれば無料枠内で十分に収まります。
- **構築の容易さ**: Dockerに慣れている場合はGCP (Cloud Run)、サーバーレス構成に慣れている場合はAWS (Lambda) が推奨されます。
